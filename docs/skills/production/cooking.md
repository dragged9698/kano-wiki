# 🍳 Cooking

> [!tip] Overview
> Cooking is an artisan skill that allows players to prepare food items. This skill is essential for creating healing items and can be a profitable way to train.

## 📋 What is Cooking?

Cooking involves:
- 🍳 Preparing raw food
- 🐟 Cooking fish
- 🥩 Cooking meat
- 🥘 Creating complex dishes
