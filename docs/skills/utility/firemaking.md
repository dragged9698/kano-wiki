# 🔥 Firemaking

> [!tip] Overview
> Firemaking is an artisan skill that allows players to create fires using logs. This skill is useful for cooking, light sources, and various other activities in the game.

## 📋 What is Firemaking?

Firemaking involves:
- 🔥 Lighting logs
- 🎨 Creating different types of fires
- 🌲 Using special logs
- 🔥 Creating bonfires
