# 🦹 Thieving

> [!tip] Overview
> Thieving is a support skill that allows players to pickpocket NPCs, steal from stalls, and complete heists. It's a great way to make money and obtain unique items.

## 📋 What is Thieving?

Thieving involves:
- 🦹 Pickpocketing NPCs
- 🏪 Stealing from stalls
- 🎯 Completing heists
- 💰 Making profit
